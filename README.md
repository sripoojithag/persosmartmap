## SmartMap Navigator

-A Location-Based Bookmark & Reminder Application that allows users to save important places and receive notifications when they are near those locations.
-The application helps users remember tasks related to specific places such as shopping, meetings, or personal errands using GPS-based location tracking.

## Features
- Location Bookmarking
Save important places directly on the map.

- Location-Based Reminders
Get notifications when you are near a saved location.

- Personalized Map View
View all bookmarked locations on a map.

- Offline Access
Access saved locations even without internet.

- Visit Planning
Plan visits by linking reminders to specific places.
